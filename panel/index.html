Anti-Backdoor — heuristik pola berbahaya di file PHP. Menangkap backdoor custom yang lolos antivirus.

          
ImunifyAV — scan signature CloudLinux. Gratis tapi tidak bisa auto-clean (v6.2+). Temuan muncul di tab ImunifyAV → gunakan Karantina panel.

          
Scan Sinergi — ImunifyAV + maldet + ClamAV + heuristik backdoor dalam satu job.

          
maldet + ClamAV — signature 51rb+ tanda tangan virus/webshell.

          
File Baru — file yang berubah dalam N hari terakhir.

          
Scan Sistem (rkhunter) — memeriksa level OS: rootkit, backdoor kernel, binary sistem yang diubah, file/proses tersembunyi. Berlaku seluruh server, bukan per-domain. Warning berlabel aman = false-positive umum OpenLiteSpeed. Otomatis setiap hari jam 03:00.

          
User WordPress — daftar user di dashboard (pilih domain). User mencurigakan ditandai merah/kuning. Nonaktifkan = turunkan ke subscriber. Karantina = kunci password + turunkan role.

        

        

        

          
≥12 Sangat berbahaya — kemungkinan webshell/backdoor

          
8-11 Mencurigakan — perlu ditinjau manual

          
6-7 Waspada — sering false-positive (library sah)

        

        

        

          
Karantina — file dipindah ke gudang aman, tidak bisa diakses/dieksekusi. Bisa dipulihkan.

          
Whitelist ON — file ditandai aman, tidak bisa dikarantina & disembunyikan dari daftar bahaya.

Scan Anti-Backdoor

Scan Malware

            
ClamAV 0.103 + Linux Malware Detect (maldet) dengan 51.000+ signature virus, trojan, dan webshell yang dikenal.

            Per-domain Background job
          

        

        

          

            

            
Deteksi File Baru

            
Temukan file yang dibuat atau diubah dalam N hari terakhir. Berguna mendeteksi upload backdoor setelah infeksi.

            Filter PHP Configurable
          

        

        

          

            

            
Scan Sistem (rkhunter)

            
Rootkit Hunter — periksa level OS: rootkit, backdoor kernel, binary sistem berubah, file tersembunyi, port mencurigakan.

            Seluruh server Cron 03:00
          

        

        

          

            

            
Karantina File

            
Isolasi file berbahaya ke gudang aman. File tidak bisa diakses web. Bisa dipulihkan atau dihapus permanen dari panel.

            Restore Hapus permanen
          

        

        

          

            

            
Whitelist File

            
Tandai file sah agar tidak diflag atau dikarantina. Berguna untuk installer.php Duplicator saat restore.

            ON/OFF toggle Per-file
          

        

        

          

            

            
Audit User WordPress

            
Daftar semua user WP per domain. Deteksi admin mencurigakan, backdoor account, admin baru. Tombol Nonaktifkan & Karantina langsung.

            Dashboard Auto-flag
          

        

        

          

            

            
Dashboard Real-time

            
Ringkasan status server: RAM, disk, load, jumlah temuan, job berjalan, user WP mencurigakan. Auto-refresh setiap 15 detik.

            Live status 16 domain
          

        

        

          

            

            
Background Jobs

            
Semua scan berjalan di background. Panel tetap responsif. Lihat log real-time, status selesai/gagal, dan riwayat job.

            Non-blocking Log viewer
